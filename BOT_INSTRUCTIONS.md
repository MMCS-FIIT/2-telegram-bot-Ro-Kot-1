# Инструкция по запуску бота для планирования тренировок

## 1. Получение токена бота

1. Откройте Telegram и найдите @BotFather
2. Отправьте команду `/newbot`
3. Придумайте имя для бота (например, "Workout Planner Bot")
4. Придумайте username для бота (должен заканчиваться на `bot`, например `my_workout_planner_bot`)
5. BotFather выдаст вам токен. Скопируйте его.

## 2. Настройка токена

Откройте файл `SimpleTGBot/WorkoutPlannerBot.cs` и замените строку:

```csharp
private const string BotToken = "ВАШ_ТОКЕН_БОТА";
```

на:

```csharp
private const string BotToken = "ВАШ_НАСТОЯЩИЙ_ТОКЕН";
```

## 3. Запуск бота

```bash
cd SimpleTGBot
dotnet run
```

## 4. Использование бота

В Telegram найдите вашего бота по username и нажмите `/start`.

Функции:
- **Добавить тренировку** - добавить новую тренировку с названием, днём недели и временем
- **Мои тренировки** - посмотреть список всех запланированных тренировок
- **Очистить все** - удалить все тренировки

## Важно

Данные хранятся только в памяти и будут сброшены при перезапуске бота.
