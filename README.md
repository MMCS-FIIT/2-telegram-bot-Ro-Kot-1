# Telegram Бот - Планировщик тренировок

## Запуск

```bash
cd SimpleTGBot
dotnet run
```

## Функции бота

- Добавление тренировок (выбор из списка или свое название)
- Просмотр запланированных тренировок
- Ведение статистики (пульс, калории, время тренировок)
- Просмотр статистики

## Токен бота

Токен уже настроен в коде (`WorkoutPlannerBot.cs`)
